# universal_youtube_player example

A demo app for `universal_youtube_player`. It shows how to load a video, switch
quality at runtime, and build custom controls on top of
`UniversalYoutubeController`.

Run it:

```bash
flutter run -d macos   # or windows, linux, chrome, or a connected device
```

The main entry point is [lib/main.dart](lib/main.dart). Integration tests that
exercise real playback live in [integration_test](integration_test).
